What Is DOM and What is the purpose of the DOM?

DOM stands For Documnet Object Model.DOM is a programming Interface for document like HTML.

Answer:

DOM stands For Document Object Model.DOM is a programming Interface for a document like HTML. The object model represents the structure of the document as an object.
Purpose Of the DOM: DOM can maintain a standard programming interface for hTML and Xml documents. Programmers can maintain a standard to create and change any document in a development environment. It creates an HTML document like a tree and also makes an object.

How Will you select HTML elements using DOM? Name the DOM Methods.

There are 5 ways in which we can select HTML elements using DOM selectors Which are:

Answer:

There are 5 ways in which we can select HTML elements using DOM selectors Which are:
1. getElementsByTagName()
2. getElementsByClassName()
3. getElementById()
4. querySelector()
5. querySelectorAll()

What is event bubble?

When two components are nested inside of one another and both elements have registered listeners for the same event,

Answer:

When two components are nested inside of one another and both elements have registered listeners for the same event, event handling can be understood as a sequence of calls to the event handlers. Thus calling is carried out starting with the most fundamental element and moving upward via all of its parents.
